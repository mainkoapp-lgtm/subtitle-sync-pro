# 향후 작업 로드맵 (Future Roadmap)

본 문서는 프로젝트의 향후 확장 계획 및 추가 기능 구현 사항을 기록합니다. 모든 작업 시 AI(안트그래비티)는 사용자의 의견 외에도 더 나은 방향이 있다면 적극적으로 제안해야 합니다.

---

## 🚀 주요 작업 목록

### 1. 다국어 지원 및 자동 브라우저 설정
- **내용**: 사용자가 접속한 국가의 언어를 감지하여 브라우저의 기본 언어를 자동으로 설정하여 열기.
- **목적**: 글로벌 사용자 편의성 증대.

### 2. 자막 추출기(Subtitle Extractor) 다운로드 기능
- **내용**: 웹 브라우저 내에 우리가 제작한 '자막 추출기'를 다운로드할 수 있는 버튼 배치.
- **링크**: FileDrop 또는 GitHub 활용 예정.

### 3. 도움말 및 사용 방법 모달(Modal)
- **내용**: 서비스 사용 방법 및 자막 추출기 사용법을 상세히 설명하는 안내 모달 제작.

### 4. 브라우저 목적 안내 모달
- **내용**: 본 서비스(브라우저 기반 도구)의 개발 목적과 취지를 설명하는 모달 제작.

### 5. 자막 추출기 소프트웨어 다국어 지원 및 운영 관리
- **다국어 지원**: 현재 개발된 자막 추출기 도구 자체도 다양한 국가의 언어를 지원하도록 수정.
- **공지 및 업데이트 관리**: 
  - 서버(JSON) 연동 공지 사항 출력 기능.
  - **버전 잠금(Version Lock)**: 특정 버전 이하의 사용을 강제로 제한하고 최신 다운로드를 유도하는 Kill Switch 기능 구현.

### 6. 서비스 수익화를 위한 광고 도입 및 운영 전략 확정
- **추출기 (EXE)**: **클릭몬(ClickMon)** 광고 탑재. (응용프로그램 매체 승인 필요)
- **웹 서비스 (Browser)**: **쿠팡 파트너스(Coupang Partners)** 매칭형 배너 탑재.
- **선행 작업**: 클릭몬 매체 등록을 위한 홍보/배포용 웹사이트(사이트 등록용) 배포 및 등록 신청.

### 7. 현재 로고는 일본어랑 영어인데, 한국어랑 영어로 변경해줘

### 8. 지능형 다국어 번역 및 서비스 차등화 (악용 방지)
- **자동 언어 감지**: Target 자막의 언어를 자동으로 감지하여 `fill_missing_subtitles`의 번역 언어를 동적으로 설정.
- **서비스 분리**: '단순 동기화'와 'AI 전체 번역' 기능을 탭으로 분리.
- **수익화/제한**: 
  - 매칭율 70% 미만일 경우 'AI 번역 탭' 이용 유도.
  - 번역 모드에서는 광고 노출을 강화하거나 별도의 비용 청구 모델 검토.
- **검증**: 원본과 대상의 분량 차이가 클 경우 API 호출 전 사용자 경고.

### 9. 보안 설계 원칙 (Security by Design) 적용
- **내용**: 배포되는 모든 결과물(Web, EXE)에 대해 기술 유출 방지 및 보안 강화 조치 적용.
- **세부 항목**:
    - **로직 격리**: 핵심 알고리즘 및 AI 프롬프트를 서버 사이드(Backend)로 격리하여 IP 보호.
    - **코드 보호**: 웹 사이트 배포 시 JS 난독화 적용, EXE 파일 배포 시 패킹 및 심볼 정보 제거.
    - **비밀 정보 관리**: API 키, DB 접속 정보 등을 소스 코드에서 제거하고 환경 변수(.env) 등으로 관리.
    - **통신 검증**: 모든 데이터 통신 암호화(HTTPS) 및 Zero Trust 기반의 서버 측 입력값 검증.

### 10. AI 캐릭터 페르소나 기반 자막 편집 기능
- **내용**: 자막 내 등장인물을 자동으로 식별하여 캐릭터별 성격(페르소나)을 부여하고, 이를 번역에 반영하는 고도화 기능.
- **핵심 기능**:
    - **캐릭터 자동 추출**: `[이름]`, `이름:` 등 패턴을 정규식으로 분석하여 유니크한 캐릭터 리스트 생성.
    - **성격(Persona) 부여 UI**: 브라우저에서 캐릭터 이름을 나열하고 사용자가 직접 개성(예: 거만함, 다정함 등)을 기재.
    - **관계도 매트릭스**: 특정 캐릭터가 타 캐릭터를 대할 때의 말투(존댓말/반말 등)를 개별 설정.
    - **시계열 성격 변화**: 자막 타임스탬프를 기준으로 특정 시간대 이후 성격이 변화하는 시나리오 지원.
    - **페르소나 프리셋**: '츤데레', '정중한 비서' 등 자주 쓰이는 말투 템플릿 제공.

### 11. 기업용 범용 번역 워크스테이션 (Enterprise Expansion)
- **영상 플레이어 통합**: 자막 편집 창 내에 오디오/비디오 지원 플레이어 탑재 및 실시간 타임라인 싱크 조정 UI.
- **범용 STT 엔진**: 장르에 상관없이 영상의 음성을 텍스트로 자동 변환(Speech-to-Text)하고 번역에 즉시 연결.
- **월드 메모리 (World-building Memory)**: 
    - **지식 자동 저장**: 번역 과정에서 확정된 명칭, 설정 등을 벡터 DB(RAG)에 자동 저장 및 후속 화수 동기화.
    - **지능형 QA**: 사용자가 번역 중 세계관이나 지난 화 설정이 헷갈릴 때 AI에게 즉석 질문 및 답변 기능.
- **일관성 검수기 (Consistency Checker)**: 전체 자막에서 고유명사 오타나 캐릭터 말투(페르소나) 어긋남을 감지하여 리포트 제공.

## 💡 작업 원칙 (AI 가이드라인)
- **적극적 의견 제시**: 모든 작업 수행 시, 사용자의 요청 사항 외에도 기술적/비즈니스적으로 더 우수한 대안이나 추가 아이디어가 있다면 항상 제안하고 논의할 것.
- **사용자 경험(UX) 우선**: 다국어 지원 및 모달 안내를 통해 처음 방문하는 사용자도 쉽게 서비스를 이용할 수 있도록 설계할 것.
