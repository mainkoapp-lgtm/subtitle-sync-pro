# GitHub 할당량 및 제한 정책

본 문서는 GitHub 공식 문서를 통해 확인한 무료 플랜(Free Plan)의 주요 할당량 및 제한 사항을 기록합니다.

## 1. 계정 단위 (Account-wide)
*계정 내 모든 저장소가 공유하는 자원입니다.*

| 항목 | 무료 플랜 할당량 | 비고 |
| :--- | :--- | :--- |
| **Actions 시간** | 월 2,000분 | 매월 1일 초기화 |
| **Actions 스토리지** | 500MB | GitHub Packages와 공유 |
| **Git LFS 스토리지** | 10 GiB | 대용량 파일 저장 공간 |
| **Git LFS 대역폭** | 10 GiB / 월 | 다운로드 시 발생하는 트래픽 |

## 2. 저장소 단위 (Per-Repository)
*각 저장소별로 독립적으로 적용되는 제한입니다.*

| 항목 | 제한 사항 | 비고 |
| :--- | :--- | :--- |
| **Actions 캐시** | 저장소당 10 GB | 빌드 속도 향상용 |
| **저장소 총 용량** | 1GB(권장) / 5GB(상한) | 초과 시 관리 대상 |
| **개별 파일 크기** | 100 MB | 50MB 초가 시 경고, 100MB 초과 시 차단 |
