# 쿠팡 파트너스 운영 및 배너 연동 지침 (2026-04-18)

## 1. 개요
본 지침서는 쿠팡 파트너스 수익 창출을 위해 홈페이지에 광고를 연동하는 방법과 정책 준수 사항을 기술합니다. 현재 단계는 **[최종 승인 전]** 상태를 고려하여 가장 안전한 '다이나믹 배너' 방식을 채택하고 있습니다.

## 2. 연동 방식: 다이나믹 배너 (Dynamic Banner)
*   **방식**: 쿠팡에서 제공하는 JavaScript 기반 스크립트를 React 컴포넌트로 래핑하여 삽입.
*   **장점**: 
    *   가입 직후(최종 승인 전) 바로 사용 가능.
    *   쿠팡 시스템이 실시간으로 상품을 갱신 (자동 업데이트).
    *   정책 위반 소지 없음.

## 3. 핵심 보안 및 정책 준수 사항 (필독)
### ① 공정거래위원회 대가성 문구
*   **필수 사항**: 본 서비스의 모든 광고 영역에는 아래 문구가 명시되어야 합니다.
*   **문구**: `이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.`
*   **구현**: `CoupangDynamicBanner.tsx` 컴포넌트 내부에 기본 포함되어 있으며, 배너 바로 아래에 가독성 있게 노출됩니다.

### ② 최종 승인 조건 (API 전환 시점)
*   **기준**: 누적 매출액 **150,000원** 달성 시 쿠팡 측의 최종 심사가 진행됩니다.
*   **API 전환**: 최종 승인 완료 후 Open API 키(Access Key, Secret Key)를 발급받아 현재의 배너를 '맞춤형 API 광고'로 업그레이드할 예정입니다.
