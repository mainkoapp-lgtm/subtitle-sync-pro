# 쿠팡 파트너스 운영 정책 및 API 이용 가이드 (2026.04 기준)

## 1. 개요
본 문서는 SubFast 및 파생 프로젝트(Web, EXE 등)에 쿠팡 파트너스 광고 및 API를 적용하기 위한 모든 정책적 제약 사항, 승인 요건, 연동 기능을 총망라하여 정리한 기술서입니다. 

## 2. 매체별 광고 게재 정책 (중요)

### [1] 웹 (Web) 서비스
* **게재 가능**: 본인 소유의 최상위 도메인 또는 서브 도메인(예: `subfast.vercel.app`)을 파트너스 '미디어 관리'에 등록한 후 승인을 받으면, 해당 웹사이트 내 어느 영역이든 광고(배너, 동적 링크 등) 노출이 가능합니다. 가장 안전한 수익화 채널입니다.

### [2] 설치형 앱 (EXE / Desktop Software)
* **직접 삽입 원칙적 금지**: 현재 쿠팡 파트너스에서는 정식 앱스토어(Play Store, App Store 등)에 등록되지 않은 **데스크톱 실행 파일(EXE/설치형 프로그램)을 등록 가능한 매체로 공식 지원하지 않습니다.**
* **리스크**: 스토어 미등록 프로그램에서 직접 호출(광고 노출 및 클릭 유도)되는 트래픽은 **부정 트래픽으로 간주되어 계정 영구 정지** 사유가 될 수 있습니다. 

#### 💡 [솔루션] 브릿지 랜딩 페이지 (Bridge Page) 전략
데스크톱 앱에서는 **직접적인 상품 링크 대신, 미리 승인받은 '공식 웹사이트'로 보내는 역할만 수행**해야 합니다.
1. 앱 안에는 "추천 상품 보기" 또는 "공지사항" 버튼을 마련.
2. 버튼 클릭 시 사용자의 기본 브라우저를 띄워 **승인 등록을 마친 우리 웹의 특정 페이지**로 이동.
3. 해당 웹 페이지(Web)에서 파트너스 광고가 노출되도록 구현하여 정책 우회 준수.

## 3. 쿠팡 파트너스 Open API V2 상세 사양

우리 계정은 이미 Access Key / Secret Key 가 발급되어 있어 즉시 연동 및 자동화가 가능합니다. 

### 3.1. 핵심 제공 기능 API
1. **상품 검색 API (`GET /v2/providers/affiliate_open_api/apis/openapi/products/search`)**
   * **기능**: 특정 키워드에 대한 실시간 쿠팡 판매 상품 리스트, 이미지, 가격 등 메타데이터 제공.
   * **활용**: AI 분석 시 자막 내용과 관련된 상품(마이크, 키보드, 영화 VOD 등)을 자동 추천.
2. **단축 채널 링크 생성 (Deeplink API)**
   * **기능**: 일반 쿠팡 상품 상세 URL을 내 트래킹 코드가 삽입된 **수익 창출용 URL**로 동적 변환.
   * **활용**: 검색 API로 찾은 상품의 링크를 수익형 링크로 변환할 때 필수.
3. **골드박스 및 카테고리 베스트 API**
   * **기능**: 일일 특가 상품 및 카테고리별 베스트셀러 내역을 가져옵니다.
   * **활용**: 랜덤하게 "오늘의 추천" 영역을 채울 때 유용.
4. **실적 리포트 API**
   * **기능**: 전일 기준 클릭 수, 구매 건수, 발생 수익금 통계를 JSON으로 전달 (매일 오후 3시 데이터 갱신).

### 3.2. 일일 API 호출 한도 및 제한 요건
* **성능 스펙**: 
  * 분당 허용 요청: **50회** (Rate limit에 걸릴 경우 일정 시간 차단됨)
  * 일 단위 허용 요청: **10,000회** 
* 시스템 설계 시, 실 실시간 호출을 남발하기보다 서버에 일시 캐싱(Cache)하거나 비동기로 호출하도록(Throttling) 설계하는 것이 바람직합니다.

## 4. 핵심 준수 및 금지 사항
* **미디어 등록 필수 조항**: 쿠팡 파트너스 링크가 클릭되는 최종 랜딩 지점(Web)은 반드시 쿠팡 파트너스 시스템 상에 등록된 도메인이어야 수익 인정.
* **법적 대가 명시 의무**: 화면 내에 반드시 **"이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다."** 문구를 노출해야 합니다.
* **백그라운드 강제 호출 금지**: 사용자 몰래 쿠키(Cookie)를 조작하거나, 웹뷰 등으로 백그라운드에 랜딩시키는 행위는 일절 금지.
