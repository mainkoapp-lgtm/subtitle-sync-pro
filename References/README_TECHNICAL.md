# Subtitle Sync Pro - 기술 지침서 및 작업 기록

본 문서는 자막 싱크 자동 맞춤 프로그램의 개발 과정, 핵심 로직, 그리고 테스트 결과를 기록하는 통합 문서입니다.

---

## 📋 프로젝트 개요
- **목적**: 싱크가 정확한 참조 자막(영어 등)을 바탕으로, 싱크가 어긋난 번역 자막(한글 등)의 시간 정보를 자동으로 교정.
- **핵심 기술**: 
  - NLP (자연어 처리) 기반 문장 유사도 분석.
  - Dynamic Time Warping (DTW) 유사 알고리즘을 통한 문장 정렬.
  - React + Vite 기반의 프리미엄 UI.

---

## 🛠 기술 스택
- **Language**: Python (Core Logic), TypeScript (Frontend)
- **Framework**: FastAPI (Backend), React (Frontend)
- **AI/NLP**: Google Gemini API (Sentence Alignment)
- **UI/UX**: Vanilla CSS (Modern & Premium Aesthetics)

---

## 🗂 데이터 구조 설계
### Subtitle Object (JSON)
```json
{
  "index": 1,
  "start": "00:00:20,000",
  "end": "00:00:24,400",
  "text": "Hello, how are you today?"
}
```

- **문제점**: 동기화 과정에서 UI의 진행률 숫자가 0%에 머물다 작업 완료 시점에 갑자기 끝나는 현상 발생.

  - 해결 로직 (main.py): `fastapi.concurrency`의 `run_in_threadpool`을 사용하여 백그라운드 스레드에서 무거운 작업을 실행하도록 분리하여 FastAPI의 논블로킹 속성 복구.
  - 진행률 세분화 (aligner.py): AI 매칭 대기 시 60%, 하이브리드 정렬 시 20% 등 단계별 진행률 업데이트 로직을 명시적으로 추가.
