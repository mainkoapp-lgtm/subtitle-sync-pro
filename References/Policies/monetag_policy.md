# Monetag 서비스 이용 정책 요약 (2026.04 확인)

## 1. 다이렉트 링크 (Direct Link / SmartLink) 정책
- **생성 개수**: 무제한 (제한 없음)
- **다중 사용**: 한 개의 웹사이트 내에서 여러 개의 다이렉트 링크를 서로 다른 위치(버튼 등)에 배치하여 사용하는 것이 허용되며, 오히려 성과 분석을 위해 권장됨.
- **권장 사항**: 트래픽 소스 및 버튼 위치별로 별도의 링크를 생성하여 성과를 추적(Tracking)하는 것을 권장함.
- **제약 사항**:
  - 링크 개수 자체에 대한 제재는 없으나, 과도한 광고 도배로 인한 사용자 경험(UX) 저해 주의.
  - 허위 클릭 유도 및 봇(Bot) 트래픽 사용 금지.
  - 트래픽의 품질이 수익금 정산의 핵심 기준임.

## 2. 수익 최적화 가이드
- 여러 개의 링크를 사용하여 페이지 내 성과가 좋은 위치(Hotspot)를 파악하고 광고 배치를 최적화할 수 있음.
